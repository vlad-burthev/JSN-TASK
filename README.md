## Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/vlad-burthev/JSN-TASK-server.git
   cd server
   npm install
   ```

2. **Create a .env file:**

   ```bash
   PORT=8888
   DB_NAME=hero
   DB_USER=postgres
   DB_PASSWORD=YOUR_PASSWORD
   DB_PORT=5432
   DB_HOST=localhost
   ```

3. **Create the database**

   ```bash
   CREATE DATABASE hero;
   ```

4. **Running the Server**

   ```bash
   npm run dev
   ```
